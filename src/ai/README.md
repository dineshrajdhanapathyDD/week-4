# AI System Module

This module contains the AI Game Director and player profiling systems.

## Components

- `AIGameDirector.ts` - Main AI decision-making system
- `PlayerProfiler.ts` - Player behavior tracking and analysis
- `AdaptiveDifficulty.ts` - Dynamic difficulty adjustment algorithms
- `ExplainableAI.ts` - AI decision logging and explanation generation

## Architecture

The AI system monitors player behavior in real-time and makes intelligent adjustments to maintain optimal challenge and engagement. All decisions are logged with human-readable explanations.
