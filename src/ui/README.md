# UI and Rendering Module

This module handles all visual rendering and user interface components.

## Components

- `RetroRenderer.ts` - Main rendering system with retro effects
- `InputHandler.ts` - Keyboard and input processing
- `VisualEffects.ts` - CRT effects, scanlines, and visual feedback
- `UIComponents.ts` - Win95-style menus and dialogs

## Architecture

The UI system provides authentic retro visuals while supporting modern features like AI decision display and adaptive visual feedback based on player performance.
